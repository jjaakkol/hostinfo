# hostinfo

`hostinfo` is a small script to display information about the host in a minimal human readable format.
It is supposed to be used in shell scripts and compute jobs.
It also  outputs Slurm job info if Slurm environment
variables are set.

```
# Host: lx4-907-31161.ad.helsinki.fi, Ubuntu 24.04.3 LTS, 6.14.0-36-generic
#  Sys: LENOVO, ThinkPad P16v Gen 1 (21FDS2VT00)
#   hw: 13th Gen Intel(R) Core(TM) i9-13900H 5200MHz, : 14 cores, 63GB mem
#  GPU: NVIDIA Corporation AD107GLM [RTX 2000 Ada Generation Laptop GPU]
#  GPU: Intel Corporation Raptor Lake-P [Iris Xe Graphics]
```
